# User Flows

## Connect a repo

1) Admin installs GitHub App
2) Platform ingests repo
3) Indexer builds artifact graph

## Track work

1) Operator opens Kanban
2) Filters to stage or `need:human`
3) Opens item -> sees linked artifacts + PRs

## Navigate knowledge

1) Open Graph
2) Search node or filter to domain
3) Click -> read markdown + see related edges
