# Organization Structure

AI-에이전트 네이티브 조직의 구조 정의.

## Teams

| Team | Pipeline Stage | Owner Role | Description |
|------|---------------|------------|-------------|
| [[management]] | Management | Operator | 우선순위, 승인, 로드맵 |
| [[research]] | Research | Researcher | 조사, 설계, 분석 |
| [[implementation]] | Implementation | Builder | 구현, 개발, 작성 |

## Pipeline Flow

```

```
